# 2018 - 2023 全国基础设施投资（不含电力、热力、燃气及水生产和供应业）

可在 [国家发改委网站](https://www.ndrc.gov.cn/) 搜索 “全国基础设施投资” 查到历史数据。
或者搜索 “全国固定资产投资” 查看 “基础设施投资” 的数据。

也可到 [国家数据网站](https://data.stats.gov.cn/) 搜索 “基础设施” 查到年度数据。

或者直接谷歌按这个格式搜索 “2022年 1-9 月 全国基础设施投资”。
